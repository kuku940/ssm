# ssm(springmvc+spring+mybatis)项目

这个项目目前作为ssm整合练手项目，熟悉整体的开发流程

## 开发环境
	服务器：tomcat7
	数据库：mysql
	开发工具：eclipse
	框架：springmvc,mybatis,spring
	包管理：maven
	版本控制：git
	
## 搭建模块

	1.用户注册和登录模块			
	2.验证码识别				
	3.邮箱发送队列					
	
	4.消息队列
	5.搜索引擎
	6.nosql数据库-mongoDB
	7.缓存系统-redis
	8.负载均衡系统
	9.
